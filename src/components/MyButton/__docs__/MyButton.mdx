# MyButton

Botón altamente personalizable que permite modificar su color, tamaño, bordes y comportamiento usando un hover.

#### Example

<Canvas of={MyButton.Primary} />

## Usage

```ts

import MyButton from 'milibreria';

function Example() {
  return (
    <MyButton
      text="Primary Button"
      txtcolor="white"
      backgroundColor="blue"
      bordercolor="blue"
      borderwidth="2px"
      hoverbackgroundcolor="darkblue"
      hovertxtColor="lightgray"
      size="medium"
      onClick={() => alert("Primary Button Clicked")}
    />
  );
}

export default Example;
